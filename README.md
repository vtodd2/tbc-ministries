# Kingdom Light Ministry

A luxury Christian ministry website built with Next.js and Tailwind CSS.

## Features

- Black and gold luxury design
- Mobile responsive layout
- SEO-friendly metadata
- Fast static export compatible with Cloudflare Pages
- Home, About, Books, Donate, Sermons, and Contact pages
- Reusable components for hero, cards, newsletter, and forms
- Contact and newsletter forms that can integrate with FormSubmit or similar services

## Setup

1. Install dependencies:

```bash
cd "~/christian-ministry-site"
npm install
```

2. Run locally:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
npm run export
```

## Deploy

This project is configured for static export and is compatible with Cloudflare Pages. Use the `out` folder generated by `npm run export` as your deploy directory.

## Notes

- Update the form action email addresses in `components/Newsletter.tsx` and `components/ContactForm.tsx` for production.
- Replace placeholder social URLs with actual ministry channels.
